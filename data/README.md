# Dataset

The dataset used in this project is **not included** in this repository.

This decision keeps the repository lightweight and ensures that users always work with the original, up-to-date version of the data.

## Primary Dataset

**Title:** Global Data on Sustainable Energy (2000–2020)

**Platform:** Kaggle

**Dataset Author:** Ansh Tanwar

This dataset contains energy, economic and environmental indicators for 176 countries over the period 2000–2020. It serves as the primary data source for the exploratory data analysis and the Machine Learning models developed in this project.

## Download

The dataset can be downloaded directly from Kaggle:

https://www.kaggle.com/datasets/anshtanwar/global-data-on-sustainable-energy

After downloading the dataset, place the CSV file inside this directory before running the notebooks.

```text
data/
└── global-data-on-sustainable-energy.csv
```
